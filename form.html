Tutorial Membuat Form Login

Berikut ini adalah contoh form login yang akan kita buat:

Penjelasan Syntax HTML:

Berikut adalah penjelasan untuk setiap elemen dalam form login:

<form>: Tag ini digunakan untuk mendefinisikan form. Atribut action dapat ditambahkan untuk menentukan URL yang akan dipanggil saat form dikirimkan.
<label>: Tag ini digunakan untuk memberikan label pada elemen input. Menggunakan atribut for menghubungkan label dengan elemen input yang sesuai.
<input>: Tag ini digunakan untuk membuat field input. Atribut type menentukan jenis input, seperti text atau password.
<button>: Tag ini digunakan untuk membuat tombol yang dapat diklik untuk mengirimkan form.
type="reset": Tombol ini mengatur ulang semua input dalam form ke nilai awalnya.
type="button": Tombol ini dapat digunakan untuk menambahkan fungsi JavaScript, seperti mengarahkan ke halaman lain.

Contoh Syntax HTML untuk Form Login:

<form>
  <div class="form-group">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required />
  </div>
  <div class="form-group">
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required />
  </div>
  <button type="submit">Login</button>
  <button type="reset">Reset</button>
  <button type="button" onclick="window.location.href='register.html';">Register</button>
</form>
