Penjelasan Syntax CSS dan Tag HTML pada Form :

border-collapse:

Menghilangkan spasi antar garis sel agar lebih rapi.

background-color:

Memberi warna latar belakang untuk membedakan bagian header dan isi tabel.

padding:

Memberikan ruang di dalam sel agar teks tidak terlalu menempel dengan border.

text-align:

Mengatur posisi teks, misalnya center agar semua isi tabel berada di tengah.

display: flex;

Menjadikan elemen fleksibel sehingga tata letaknya bisa disesuaikan dalam satu baris atau kolom.

justify-content:

Mengatur bagaimana konten di dalam elemen fleksibel didistribusikan secara horizontal, misalnya "center" untuk membuat konten di tengah.

align-items:

Mengatur penyelarasan vertikal elemen fleksibel. "center" akan membuat elemen berada di tengah secara vertikal.

width:

Mengatur lebar elemen, bisa menggunakan persentase atau satuan piksel (px).

border-radius:

Membuat sudut elemen melengkung. Semakin besar nilainya, semakin melengkung sudut elemen.

box-shadow:

Memberikan efek bayangan pada elemen. Parameter pertama adalah horizontal offset, yang kedua vertical offset, dan yang ketiga adalah tingkat blur.

<form>:

Mendefinisikan form untuk mengumpulkan input pengguna. Atribut action menentukan URL pengolahan data.

<input>:

Menghasilkan elemen interaktif untuk pengambilan input pengguna. Tipe input seperti text dan password digunakan untuk menerima data yang sesuai.

<label>:

Mendefinisikan label untuk elemen form. Atribut for mengaitkan label dengan input yang sesuai.

<button>:

Digunakan untuk membuat tombol dalam form. Tombol dapat digunakan untuk melakukan aksi tertentu seperti mengirimkan form.

<div>:

Elemen kontainer umum yang dapat digunakan untuk mengelompokkan elemen lain, sering digunakan dengan CSS untuk pengaturan tata letak.
