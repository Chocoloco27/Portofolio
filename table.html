Penjelasan Syntax CSS dan Tag HTML pada Tabel:

border-collapse: Menghilangkan spasi antar garis sel agar lebih rapi.

background-color: Memberi warna latar belakang untuk membedakan bagian header dan isi tabel.

padding: Memberikan ruang di dalam sel agar teks tidak terlalu menempel dengan border.

text-align: Mengatur posisi teks, misalnya center agar semua isi tabel berada di tengah.

<table> - Elemen utama untuk membuat sebuah tabel di HTML. Semua baris dan kolom tabel ditempatkan di dalam elemen ini.

<thead> - Elemen ini digunakan untuk mendefinisikan bagian kepala atau header dari tabel. Biasanya berisi judul kolom yang diletakkan di bagian atas tabel.

<tbody> - Elemen ini digunakan untuk menampung isi tabel atau konten utama. Data yang ada di dalam tabel ditempatkan di dalam elemen (tbody).

<th> - Elemen ini digunakan untuk membuat header (judul) di kolom tabel. Teks yang diletakkan di dalam elemen (th) secara default ditampilkan dengan teks tebal dan di tengah-tengah sel.

<tr> - Digunakan untuk mendefinisikan satu baris dalam tabel. Setiap baris di dalam tabel ditempatkan di dalam elemen tr.

<td> - Elemen ini digunakan untuk mendefinisikan satu sel data di dalam tabel. Setiap data dalam tabel ditempatkan di dalam elemen (td). Ini adalah elemen yang berisi konten data pada setiap baris tabel.



Berikut Ini syntax Html yang bisa anda coba:




<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Tutorial Membuat Table</title>
    <style>
      /*Styling untuk tabel*/
      table {
        width: 100%;
        border-collapse: collapse; /* Menggabungkan border antara sel */
      }
      th,
      td {
        border: 1px solid black; /* Menambahkan border pada setiap sel */
        padding: 8px;
        text-align: left; /* Menyelaraskan teks ke kiri */
      }
      th {
        background-color: #2db7e5; /* Memberikan warna latar pada header tabel */
      }
    </style>
  </head>
  <body>
    <h2>Tutorial Membuat Tabel HTML</h2>

    <table>
