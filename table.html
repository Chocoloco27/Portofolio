Penjelasan Syntax CSS dan Tag HTML pada Tabel:

<table>: Tag ini digunakan untuk mendefinisikan tabel di HTML. Semua elemen tabel, termasuk baris dan sel, harus berada di dalam tag ini.

<thead>: Tag ini digunakan untuk mengelompokkan bagian header dari tabel, yang biasanya berisi judul kolom.

<th>: Tag ini digunakan untuk mendefinisikan sel header dalam tabel. Teks di dalam tag ini biasanya akan dicetak tebal dan dipusatkan secara default.

<tbody>: Tag ini digunakan untuk mengelompokkan konten utama tabel, yaitu data yang ditampilkan dalam bentuk baris dan kolom.

<tr>: Tag ini digunakan untuk mendefinisikan baris dalam tabel. Baik dalam bagian header (<thead>) maupun konten (<tbody>), baris-baris data didefinisikan dengan tag ini.

<td>: Tag ini digunakan untuk mendefinisikan sel data dalam baris tabel. Ini adalah tempat Anda akan menempatkan nilai atau informasi.

border-collapse: Mengatur cara batas antar sel ditampilkan. Nilai collapse akan membuat batas-batas sel saling menyatu, memberikan tampilan yang lebih bersih.

padding: Menambahkan ruang di dalam sel antara isi sel dan batas sel. Ini memberikan sedikit jarak agar informasi lebih mudah dibaca.

text-align: Mengatur perataan teks dalam sel. Nilai left membuat teks diratakan ke kiri.

background-color: Mengatur warna latar belakang sel header untuk membuatnya lebih menonjol dibandingkan sel biasa.


Berikut Ini syntax Html yang bisa anda coba:


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Tutorial Membuat Table</title>
    <style>
      /*Styling untuk tabel*/
      table {
        width: 100%;
        border-collapse: collapse;
      }
      th, td {
        border: 1px solid black;
        padding: 8px;
        text-align: left;
      }
      th {
        background-color: #2db7e5;
      }
    </style>
  </head>
  <body>
    <h2>Tutorial Membuat Tabel HTML</h2>
    <table>
      <thead>
        <tr>
          <th>NIM</th>
          <th>Nama</th>
          <th>IPK Semester 1</th>
          <th>IPK Semester 2</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td>12345678</td>
          <td>Siti Zulekha</td>
          <td>3.8</td>
          <td>3.9</td>
        </tr>
      </tbody>
    </table>
  </body>
</html>
