Tutorial Deployment ke Github

Langkah 1: Buat Repositori GitHub

1. Buka GitHub dan login ke akun Anda.

2. Klik tombol New untuk membuat repositori baru.

3. Berikan nama repositori (misalnya my-tutorial-website).

4. Setel repositori ke Public dan klik Create repository.

Langkah 2: Upload Kode Website Anda ke Repositori

Ada dua cara untuk mengunggah file:

  • Upload langsung di GitHub:

    • 1. Di dalam repositori baru, klik Add file > Upload files.

    2. Seret dan jatuhkan file HTML, CSS, JavaScript, dan gambar Anda ke dalam repositori.

    3. Setelah selesai, klik Commit changes untuk mengunggah file.

  • Upload menggunakan Git di terminal:

    • 1. Buka terminal dan arahkan ke direktori proyek Anda.

    2. Jalankan perintah berikut untuk menginisialisasi Git, menambahkan file, dan mendorongnya ke GitHub:

    git init
    git add .
    git commit -m "Initial commit"
    git branch -M main
    git remote add origin https://github.com/username/my-tutorial-website.git
    git push -u origin main

Langkah 3: Aktifkan GitHub Pages

1. Buka tab Settings di repositori GitHub Anda.

2. Gulir ke bawah ke bagian GitHub Pages.

3. Pada opsi Source, pilih main dan folder / (root).

4. Klik Save.

Setelah itu, URL untuk website Anda akan terlihat di bagian GitHub Pages. URL biasanya seperti https://username.github.io/my-tutorial-website.

Langkah 4: Cek Website Anda

Buka URL yang dihasilkan untuk melihat website Anda yang sudah online!
